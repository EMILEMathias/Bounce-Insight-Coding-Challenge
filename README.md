# Bounce-Insight-Coding-Challenge

## To setup the project :

### Setup back

Install and start :

```
cd backend
npm install
npm run start
```

### Setup front

Create a .env file with the backend API endpoint such as :

```
REACT_APP_BACKEND_URL=http://localhost:3001/country/
```

Then install and start :

```
cd ../frontend
npm install
npm run start
```
